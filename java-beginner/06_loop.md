6. 반복문
- while문1

while(조건) 형태로 같은 코드를 반복적으로 실행이 가능하여 간편하다.

- while문2

반복되는 와중 변수의 값을 점점 늘릴수 있다. 조건을 잘 정해서 반복되는 횟수를 원하는 만큼 설정하는 것이 핵심!

- do-while문

while문으로 반복하는 도중에 무언가를 실행하고 싶을 때 사용한다. 최초 1번은 실행된다.

- break, continue

break는 while문을 아예 끝내버리는 것이고continue는 while문을 다음 반복으로 바로 넘어가게 해준다.(건너뛰기)

while(true)는 무한반복이다.

- for문1

for(초기식 ; 조건식 ; 증감식) 형태이다. C,C++에서 배운것과 동일.

- for문2

초기 설정을 해주고 정해진 조건 내에서 증감을 하면서 계속 반복해서 코드를 실행해준다. 

- 중첩 반복문

Nested구조

반복문 안에 반복문을 넣음으로써 i와j을 이용해서 2차원 구조처럼 만들 수 있다. 행렬이 필요할 때나 구분을 지어야 할 때 유용하다.
