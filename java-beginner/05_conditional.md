5. 조건문
- if문1 - if, else

if( )의 형태이며 괄호 안에 조건을 넣으면 True 와 False의 값을 판별하고, True인 경우에 뒤에 따라오는 문장을 실행한다.

False인 경우에 무언가를 실행하려면 else를 이용하면 된다. 이지선다의 경우 if와 else의 두가지 경우로 나누기 쉽다.

- if문2 - else if

if만으로 반복하여 코드를 완성해도 된다. 그렇지만 이미 조건을 달성했음에도 계속 조건 검사를 하게 된다는 단점이 있다. 또한 이미 검사를 한 조건을 통해 추정할 수 있는 내용들을 간과한다는 비효율적인 면이 있다. 이럴때 else if를 이용하는 것이 권장된다.

- if문3 - if와 else if

각각 실행하면 if만을, 연관이 있으면 else if를 사용한다.

- switch

if와 else if를 반복하는 대신 case로 나누어 실행문을 정하는 방법이다. break;를 꼭 사용해주어 끊어주어야 한다.

- 삼항 연산자

(조건) ? 참일때 실행문 : 거짓일때 실행문

다음과 같은 구조를 가진다. 괄호안의 내용이 참이면 전자, 거짓이면 후자의 결과값을 가진다.
